Textarea

The textarea element represents a multi-line plain-text editing control, useful when you want to allow users to enter a sizeable amount of free-form text, for example a comment on a review or feedback form.

Attribute - cols="20" (default)

The default value. A single-line text field. Line-breaks are automatically removed from the input value. 

<textarea name="textarea" rows="5" cols="20" placeholder="Textarea">Text goes here </textarea>

Attribute - rows="10"

A multi-line text field. Line-breaks are automatically removed from the input value. The cols attribute specifies the number of characters per line. The rows attribute specifies the number of lines. 

                
<textarea name="textarea"  cols="45" rows="10" placeholder="Textarea">Text goes here </textarea>
